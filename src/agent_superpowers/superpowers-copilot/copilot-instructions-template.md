# Copilot Instructions — Superpowers

You have access to a library of workflow skills.

## Available Skills

Skills are in `src/agent_superpowers/superpowers-copilot/skills/`. Load any skill with:
`#file:src/agent_superpowers/superpowers-copilot/skills/<skill-name>/SKILL.md`

## When to Use Which Skill

- **New feature or component** → load `brainstorming/SKILL.md` first
- **Have a spec, ready to plan** → load `writing-plans/SKILL.md`
- **Have a plan, ready to implement** → load `executing-plans/SKILL.md`
- **Bug or unexpected behavior** → load `systematic-debugging/SKILL.md`
- **About to write code** → load `test-driven-development/SKILL.md`
- **About to claim work is done** → load `verification-before-completion/SKILL.md`
- **Received code review feedback** → load `receiving-code-review/SKILL.md`

## Rule

If there is even a 1% chance a skill applies to what you are doing, load and follow it.
